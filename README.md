### KEY FEATURES
* User signup
* User login
* Reset password
* Account verification & activation
* Record overview dashboard
* Material recording and Update
* Material views
* Daily material usage view and analyses
* Equipment recording, updates & views
* Notification via e-mail on events
* General statistics


### INSTALLATIONS

#### PORT => 2020

In the project directory, run ``` npm init ``` to initiate and customize the project:
```
npm init
```

For dependency installations, use:
```
npm install <dependencyName>
```

For dependency vulnerabilities fixes, 
run `npm audit` for details or `npm audit fix` to fix them.
```
npm audit
npm audit fix
```

Configure mongoDB on your local machine or use Atlas

Connect your DB url and fire up

Open http://localhost:2020 to view in the browser.
